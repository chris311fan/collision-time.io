Collision Time

A private Discord bot for competitive League of Legends teams — automating draft management, post-game analysis, and opponent scouting for organized scrims.

🎮 Riot API Compliant 💬 Discord Bot ✅ Free & Non-Commercial

What It Does

Built for Competitive Teams

Collision Time Bot streamlines the entire scrim night workflow — from scouting opponents before the match, to managing drafts, to generating detailed post-game reports. All from within Discord.

🔍

Opponent Scouting

Input summoner names or paste an op.gg multi-search URL. The bot generates detailed player profiles with ranked stats, champion pools, mastery data, playstyle analysis, and exploitable weaknesses — plus a team summary with suggested bans.

⚔️

Draft Management

Create draft series via integration with Drafter.lol. The bot generates shareable draft room links, creates organized threads, polls for completed drafts, and posts pick/ban results with champion images automatically.

📊

Post-Game Reports

Automatically detects when linked players finish a game and generates a detailed post-game scorecard image with KDA, CS, damage, vision, gold, items, objectives, and team comparison — posted directly into the series thread.

🔗

Player Linking

Team members link their Riot accounts to Discord via slash command. The bot resolves PUUIDs and monitors linked players for automatic game detection and post-game report generation.

🏢

Multi-Server Support

The bot supports multiple Discord servers simultaneously, each with independent configuration — team branding, channel mapping, logo, colors, and thread prefixes. Server admins configure via /setup commands.

🧠

Playstyle Intelligence

Analyzes last 10 ranked games per scouted player to identify playstyle tags (Farm Heavy, Team Fighter, Carry Threat, Low Vision), comfort vs. new champion picks, and rank discrepancies between Solo and Flex queues.

How It Works

Scrim Night User Flow

The typical user journey from scouting an opponent to reviewing post-game results — all within Discord.

1

Link Player Accounts

Team members link their Riot accounts to enable automatic game detection and post-game reports.

/link player:Name#TAG region:NA
2

Scout the Opponent

Enter summoner names or an op.gg URL. The bot fetches ranked stats, recent match history, champion mastery, and generates individual scouting profiles plus a team summary with ban suggestions.

/scout players:Name1#TAG, Name2#TAG team:OpponentName
3

Create a Draft Series

Start a Best-of series. The bot creates a draft room, opens a thread in #scrim-results, and posts the link. If a scout thread exists for that opponent, it reuses it.

/draft opponent:OpponentName games:3 side:blue
4

Draft Results Auto-Posted

The bot polls the draft room for completion and automatically posts each game's pick/ban results with champion splash images into the series thread.

5

Post-Game Report Generated

After the in-game match finishes, the bot detects it via the Riot Spectator/Match API and generates a full scorecard image — KDA, CS, damage, vision, gold, items, and objectives — posted in the thread.

6

Series Complete

After all games are drafted, the bot posts a series summary with overall results. The thread serves as a permanent archive of the entire scrim: scout report → drafts → post-game results.

Technical Details

Riot API Usage

The bot uses the following Riot Games API endpoints. All data is fetched read-only and used solely for in-Discord display. No data is stored persistently, sold, or shared.

Endpoint Purpose Access
riot/account/v1 Resolve Riot ID (gameName#tagLine) to PUUID for API lookups READ
lol/summoner/v4 Fetch summoner level and profile icon for scouting display READ
lol/league/v4 Fetch ranked tier, division, LP, wins, and losses for scouting READ
lol/champion-mastery/v4 Fetch top champion masteries to analyze comfort picks and pool depth READ
lol/match/v5 Fetch recent match IDs and match details for performance analysis and post-game reports READ
lol/spectator/v5 Detect when linked players are currently in-game to trigger post-game monitoring READ
Data Dragon Champion names, profile icons, champion splash images (static assets) STATIC

Compliance

Privacy & Riot Policy Compliance

Collision Time Bot is built with Riot Games' developer policies and player privacy as top priorities.

✅ No Data Storage

Player data is fetched on-demand and displayed in Discord only. No match history, ranked stats, or personal information is stored in any database or persisted to disk.

✅ No De-anonymization

The bot does not expose historic Riot IDs, track players across name changes, or reveal any information that isn't already publicly visible on a player's profile.

✅ Read-Only Access

All API usage is strictly read-only. The bot never modifies, writes to, or interacts with any game client, player account, or Riot service beyond fetching public data.

✅ No Monetization

This is a free, non-commercial tool built for private team use. There is no subscription, no paid tier, no advertising, and no data selling of any kind.

✅ Rate Limit Compliance

The bot implements proper rate limiting with request queuing, Retry-After header handling, and exponential backoff. All requests stay well within Riot's published rate limits.

✅ No Game Interference

The bot operates entirely outside the game client. It does not inject code, modify game files, automate gameplay, or interact with the League of Legends client in any way.
